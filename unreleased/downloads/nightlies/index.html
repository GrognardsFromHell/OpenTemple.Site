Nightly Builds

We provide fully automated builds of the latest OpenTemple code for testing and development purposes.

Please keep in mind that these builds may be outright broken or corrupt your save games. In case you have any valuable save data, please back it up before using a nightly build!

Windows 10

Last Update: 01 Jan 2023 19:44 UTC

The nightly builds for Windows 10 are provided as Windows App packages which allow for smooth automated updates and clean removal.

We recommend using the automated installer for OpenTemple nightly builds. It will automatically install the self-signed certificate required to install the App.
